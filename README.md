# blogstar
This is basic blog application in Spring Boot and Angular

**Functionalities**
1. Registration
2. Login/Logout
3. Create new blog post
4. View blog posts
5. User verification using JWT token

**Technologies:**
1. Spring Boot 2.2.6.RELEASE
2. Angular
3. MySql
4. JWT
